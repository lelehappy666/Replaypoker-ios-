# 牌桌筹码联动、主页面稳定与音效实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让筹码堆随筹码飞行动画同步增减，彻底稳定四个主页面布局，压缩右上账户余额区，并加入可持久关闭的牌桌音效。

**Architecture:** 四个主页面使用常驻页面栈与像素对齐固定外壳；牌桌展示层通过事件和动画进度推导临时金额，不修改牌局数据；音效由独立本地服务根据牌桌事件播放程序生成的短音频。所有新增设置使用统一键值持久化。

**Tech Stack:** Swift 6、SwiftUI、Observation、AVFoundation、XCTest、Swift Testing。

## Global Constraints

- 所有说明文档和 Git 提交信息使用中文。
- 直接在 `main` 开发，不创建额外工作树。
- iPhone 16 Pro Max 全程横屏。
- 开启系统“减少动态效果”时仍保留分层筹码飞行和堆高变化。
- 不增加大规模 UI 自动化测试，只做直接相关的逻辑测试与编译验证。
- 音效默认开启，可在个人中心持久关闭，不覆盖系统静音策略。
